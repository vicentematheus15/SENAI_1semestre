# 📚 SENAI (1º Semestre)

Este repositório reúne **exercícios e projetos** realizados durante o **primeiro semestre** do curso técnico de **Desenvolvimento de Sistemas** no SENAI.  
Ele reflete minha evolução inicial na área de programação, eletrônica e fundamentos para o desenvolvimento web.

---

## 📖 Sobre este semestre

Durante este primeiro semestre, tive contato com diferentes áreas fundamentais para a formação como desenvolvedor:

- **Fundamentos da programação web** com **HTML**, **CSS** e **JavaScript**.
- **Lógica de programação**, praticada inicialmente com **Portugol** e **Python** para fixar conceitos.
- Introdução ao **C++** utilizando **Arduino** no **Tinkercad** na matéria de **Eletroeletrônica**.
- Noções iniciais de **Programação Orientada a Objetos (POO)**.

Todos os arquivos deste repositório correspondem a **exercícios e projetos feitos em aula**, geralmente com apoio dos professores.  
Os projetos avaliativos feitos individualmente não estão neste repositório.
